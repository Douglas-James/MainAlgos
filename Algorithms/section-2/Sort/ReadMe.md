# Sorting Algorithms

In this folder, we focus on various sorting algorithms, categorized by their performance:

- **Slow Algorithms**:

  - Bubble Sort
  - Insertion Sort
  - Selection Sort

- **Medium Algorithms**:

  - Merge Sort
  - Merge Array

- **Fast Algorithms**:

  - Quick Sort (with pivot helpers)
  - Radix Sort (with radix helpers)

Happy Coding!
